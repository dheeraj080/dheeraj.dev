A meticulously crafted Nix-based development environment that ensures reproducibility across multi-platform teams. Includes optimized dotfiles and containerized dev environments.

## Setup Highlights
- **Reproducible Builds:** Leveraging Nix to ensure every developer has the exact same toolchain versions.
- **Containerization:** Pre-configured Docker environments for database and service mocks.
- **Editor Optimization:** Custom Neovim configuration tailored for Full Stack Engineering (LSP, Tree-sitter, etc.).
- **Dotfiles Automation:** One-command setup for new machines using shell scripts and symlinking.
