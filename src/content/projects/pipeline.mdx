Next-gen data ingestion engine capable of processing millions of events per second with sub-millisecond latency. Built with Kafka for messaging and K8s for orchestrating workers.

## Technical Details
- **High Throughput:** Optimized Kafka producer/consumer groups to handle bursty workloads.
- **Auto-scaling Workers:** Kubernetes Horizontal Pod Autoscalers (HPA) scale processing power dynamically.
- **Data Integrity:** Implemented exactly-once processing semantics to prevent data loss or duplication.
- **Monitoring:** Integrated Prometheus and Grafana for real-time observability of pipeline health.
