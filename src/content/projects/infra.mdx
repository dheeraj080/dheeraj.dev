A robust automation suite for provisioning highly available AWS infrastructure using Terraform and Go. Includes custom provider support and automated scaling policies.

## Key Features
- **Custom Terraform Providers:** Built modular providers for proprietary internal services.
- **Automated Scaling:** Implemented dynamic scaling group policies based on real-time traffic metrics.
- **Security First:** Integration with AWS IAM and Secrets Manager for zero-trust environment setup.
- **Continuous Deployment:** Full CI/CD pipelines using GitHub Actions for infrastructure as code.
